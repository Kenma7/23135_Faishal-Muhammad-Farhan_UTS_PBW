Perulangan dalam PHP

Apa Itu Perulangan?

Perulangan (looping) digunakan untuk menjalankan blok kode berulang kali selama kondisi tertentu masih terpenuhi.

Jenis Perulangan dalam PHP

1. Perulangan for

Digunakan ketika jumlah iterasi sudah diketahui.

<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Angka: $i <br>";
}
?>

2. Perulangan while

Digunakan ketika jumlah iterasi tidak diketahui dan bergantung pada suatu kondisi.

<?php
$i = 1;
while ($i <= 5) {
    echo "Angka: $i <br>";
    $i++;
}
?>

3. Perulangan do-while

Mirip dengan while, tetapi akan dijalankan minimal satu kali.

<?php
$i = 1;
do {
    echo "Angka: $i <br>";
    $i++;
} while ($i <= 5);
?>

4. Perulangan foreach (Untuk Array)

Digunakan untuk melakukan iterasi pada elemen array.

<?php
$buah = ["Apel", "Mangga", "Jeruk"];
foreach ($buah as $item) {
    echo "Buah: $item <br>";
}
?>
