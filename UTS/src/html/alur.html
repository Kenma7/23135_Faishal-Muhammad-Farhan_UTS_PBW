Apa itu Pengkondisian?

Pengkondisian dalam pemrograman adalah cara untuk mengontrol jalannya program berdasarkan kondisi tertentu.

Jenis Pengkondisian dalam PHP

  • If Statement - Mengeksekusi blok kode jika kondisi bernilai true.
  • If-Else Statement - Menjalankan kode alternatif jika kondisi tidak terpenuhi.
  • Elseif Statement - Digunakan untuk mengecek beberapa kondisi secara berurutan.
  • Switch Statement - Digunakan ketika ada banyak kemungkinan kondisi.

Contoh Implementasi

Berikut adalah contoh sederhana penggunaan pengkondisian dalam PHP:

Contoh If-Else

<?php
$nilai = 75;

if ($nilai >= 80) {
    echo "Selamat, Anda mendapatkan nilai A!";
} elseif ($nilai >= 70) {
    echo "Anda mendapatkan nilai B.";
} else {
    echo "Anda perlu belajar lebih giat.";
}
?>
