Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas ke dalam halaman web, seperti animasi, validasi form, dan event handling.

Cara Menggunakan JavaScript

Ada tiga cara menambahkan JavaScript ke dalam halaman web:

  • Inline JavaScript: Ditulis langsung dalam atribut elemen HTML.
  • Internal JavaScript: Ditulis dalam tag <script> di dalam file HTML.
  • External JavaScript: Menggunakan file .js terpisah.

Contoh Implementasi

Berikut adalah contoh sederhana penggunaan JavaScript untuk menampilkan pesan saat tombol ditekan:

Contoh JavaScript

<button onclick="tampilkanPesan()">Klik Saya!</button>

<script>
function tampilkanPesan() {
    alert("Halo! Ini adalah contoh JavaScript.");
}
</script>
