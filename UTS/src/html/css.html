Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan halaman web, seperti warna, font, dan layout.

Cara Menggunakan CSS

Ada tiga cara menggunakan CSS dalam HTML:

  • Inline CSS: Langsung di dalam elemen HTML menggunakan atribut style.
  • Internal CSS: Ditulis dalam tag <style> di dalam bagian <head>.
  • External CSS: Menggunakan file terpisah dengan ekstensi .css.

Contoh Implementasi

Berikut adalah contoh penggunaan CSS untuk mengubah warna teks dan latar belakang:

Contoh CSS

/* CSS Eksternal */
body {
    background-color: #1a1a2e;
    color: white;
    font-family: Arial, sans-serif;
}

/* CSS Internal */
<style>
h1 {
    color: #6a5acd;
}
</style>

/* CSS Inline */
<h1 style="color: red;">Teks Merah</h1>
