Array dalam PHP

Apa Itu Array?

Array adalah struktur data yang dapat menyimpan banyak nilai dalam satu variabel di PHP.

Jenis-Jenis Array

1. Array Numerik

Array yang menggunakan indeks angka secara otomatis.

<?php
$buah = ["Apel", "Mangga", "Jeruk"];
echo $buah[0]; // Output: Apel
?>

2. Array Asosiatif

Array dengan kunci (key) yang dapat ditentukan sendiri.

<?php
$umur = ["Andi" => 20, "Budi" => 22, "Citra" => 21];
echo $umur["Budi"]; // Output: 22
?>

3. Array Multidimensi

Array di dalam array.

<?php
$mahasiswa = [
    ["Nama" => "Andi", "Umur" => 20],
    ["Nama" => "Budi", "Umur" => 22]
];
echo $mahasiswa[1]["Nama"]; // Output: Budi
?>
